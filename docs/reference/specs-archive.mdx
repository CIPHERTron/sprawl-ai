The `specs/` directory at the repo root contains the complete design archive for Sprawl AI. Each phase was approved before implementation of that milestone began.

<Info>
  These documents explain **why** decisions were made. For **what is currently implemented and how to work with it**, use the pages in this documentation site. The two are complementary — specs cover design rationale; docs cover runtime reality.
</Info>
